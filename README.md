Tributary System – Honoring Heroes of the Indian Armed Forces 

A full-stack web application that allows users to create, view, and share tributes for heroes of the Indian Armed Forces.
Users can upload images, write stories about soldiers, and honor their bravery on a public platform.
This project demonstrates backend API design, authentication, file uploads, and full-stack integration using React and Node.js.

Features
User registration and login (JWT Authentication)

Secure authentication using JSON Web Tokens

Create tribute posts with image upload

View tributes posted by different users

Edit and delete tributes (only by the creator)

Responsive card layout displaying hero tributes

Modal-based UI for login and tribute creation
